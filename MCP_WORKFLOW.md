## 🔄 详细工作流程

### 场景 1: 使用预设模板（无 AI）

```
用户: "渲染一个移动端登录页"
    │
    ▼
MCP Tool: render_ui
    │
    ├── 参数: { template: "mobile-login", width: 375, height: 812 }
    │
    ▼
选择模板 → leafer-x-design-system.js
    │
    ├── 获取 mobile-login 模板配置
    ├── 创建 Leafer 画布 (375x812)
    ├── 绘制背景、Logo、输入框、按钮
    └── 应用预设样式
    │
    ▼
渲染服务 (Port 3456)
    │
    ├── 接收渲染请求
    ├── 调用 LeaferRendererV2.render()
    ├── 生成 PNG 图片
    └── 保存到 output/ 目录
    │
    ▼
返回结果给 Trae
    │
    └── 用户看到登录页原型图
```

### 场景 2: 使用 AI 生成（配置生图模型后）

```
用户: "帮我设计一个现代化的电商首页，参考淘宝风格"
    │
    ▼
MCP Tool: generate_ai_ui (假设已添加)
    │
    ├── 参数: { 
    │     prompt: "现代化的电商首页，参考淘宝风格",
    │     style: "现代简洁",

    │
    ▼

    ├── 1. 构建 System Prompt
    │   "你是一个专业的 UI/UX 设计师..."

    │   "风格要求: 现代简洁"
    │   "主色调: #ff6900"
    │
    ▼

    │
    ├── 国内模型选择:
    │   ├── 通义千问: https://dashscope.aliyuncs.com/...

    ├── 国外模型选择:
    │   ├── OpenRouter: https://openrouter.ai/...

    │   └── ...
    │
    ▼
AI 返回 UI 描述 (JSON)

            "logo": "淘宝",

        {
          "type": "banner-carousel",
          "props": {
            "images": ["banner1", "banner2", "banner3"],
            "autoPlay": true
          }

        {
          "type": "category-grid",
          "props": {
            "categories": ["女装", "男装", "数码", "家居", ...]
          }
        },
        {
          "type": "product-list",
          "props": {
            "title": "猜你喜欢",

JSON → Leafer 元素转换
```

## 🔧 配置示例

### 1. 配置 AI 模型（环境变量）

```bash
# .env 文件
# 国内模型
QWEN_API_KEY=sk-your-qwen-api-key
DOUBAO_API_KEY=your-doubao-api-key
DEEPSEEK_API_KEY=sk-your-deepseek-api-key

# 国外模型
OPENROUTER_API_KEY=sk-or-your-openrouter-key
ANTHROPIC_API_KEY=sk-ant-your-claude-key

# 默认配置
DEFAULT_AI_PROVIDER=qwen
DEFAULT_MODEL=qwen-max
```

### 2. MCP 配置

```json
{
  "mcpServers": {
    "leafer-design-system": {
      "command": "node",
      "args": ["${workspaceFolder}/mcp-server.js"],
      "env": {
        "QWEN_API_KEY": "${env:QWEN_API_KEY}",
        "DEEPSEEK_API_KEY": "${env:DEEPSEEK_API_KEY}"
      }
    }
  }
}
```

### 3. 启动服务

```bash
# 终端 1: 启动 MCP 服务
npm start
# 或
node start-mcp-service-v2.js

# 服务启动后监听端口 3456
# 等待 Trae 的 MCP 调用
```

## 📋 数据流总结

| 步骤 | 组件 | 输入 | 输出 |
|------|------|------|------|
| 1 | 用户 | 自然语言描述 | - |
| 2 | Trae MCP Client | Tool 调用请求 | JSON 参数 |
| 3 | MCP Server | Tool 名称 + 参数 | 处理指令 |
| 4 | AI Generator | Prompt + API Key | UI JSON 描述 |
| 5 | Leafer Renderer | UI JSON | 矢量图形对象 |
| 6 | Render Service | 图形对象 | PNG 图片 |
| 7 | Trae | 图片路径 | 显示给用户 |

## ⚡ 关键优势

1. **灵活性**: AI 生成任意风格的 UI，不局限于预设模板
2. **可编辑**: 返回的 JSON 描述可以被修改后再渲染
3. **高质量**: LeaferJS 矢量渲染，支持高分辨率输出
4. **多模型**: 支持国内外 16+ 种 AI 模型，可自由选择
5. **可扩展**: 易于添加新的 AI 提供商或渲染引擎
